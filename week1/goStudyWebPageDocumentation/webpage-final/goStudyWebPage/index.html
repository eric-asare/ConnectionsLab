Motivation

Do you love Social Media as I do and cannot get work or studies done? You are at the right place.
goStudy blocks all social media websites on a single click!

Development Process

This extension is essentially a collection of HTML, CSS, and JavaScript codes that allow us to add some extra functionality to the Chrome browser using its JavaScript APIs.

When the goStudy extension detects a distracting website like instagram from the window location hostname, it redirects the user to the default page.

The Coding Process

To build the extension , I needed

  • An icon for the extension on all devices
  • A manifest file that tells Chrome what needs to be loaded so that it runs effectively
  • A user interface that is displayed when the extension icon is clicked
  • The Javascript(logic) to detect url visited and redirect the user
  • The redirect page to inspire user to keep studying
